# Sonoran Roots — brand assets

Public-facing brand assets (logos, wordmarks). Public so they can be referenced
by HTTPS URL from third-party tools like Omni Analytics theme settings.

## Files

| File | Use |
| --- | --- |
| `omni-logo.png` | Square logo (24×24px slot in Omni Settings → Themes → Application → Branding assets) |
| `omni-wordmark.png` | Horizontal wordmark (100×24px slot) |

## Direct URLs

Use the `raw.githubusercontent.com` URLs (the `github.com/.../blob/...` URLs serve HTML, not the raw image):

- Logo: `https://raw.githubusercontent.com/Sonoran-Roots/brand-assets/main/omni-logo.png`
- Wordmark: `https://raw.githubusercontent.com/Sonoran-Roots/brand-assets/main/omni-wordmark.png`
